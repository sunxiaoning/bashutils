# bashutils
provide some useful bash utils

## yum installer
see `yuminstaller.sh`.

### function
provide `yum install with version` .

### how to use
use command `yuminstaller.sh -h` to find usage.

### about option
use `-f` option to ignore old version already exists (allow version upgrade or override).
